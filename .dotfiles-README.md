# Dotfiles config tracker

## Setup

Clone the repository to your user root with

```bash
git clone --bare https://github.com/julianhamre/dotfiles $HOME/.dotfiles
```

Declare the dotfiles alias by running

```bash
alias dotfiles='git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME'
```

This alias will be introduced in .zshrc later. 

Hide untracked files with

```bash
dotfiles config status.showUntrackedFiles no
```

Thereafter, introduce this repo's config files with

```bash
dotfiles checkout 
```

If the checkout is unsuccessful because some of the config files already exist on your machine,
back them up and rerun the checkout command.

Finally, reload your shell with

```bash
source ~/.zshrc
```

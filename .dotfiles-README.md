# Dotfiles config tracker

## Setup

Clone the repository to your user root with

```bash
git clone --bare https://github.com/julianhamre/dotfiles $HOME/.dotfiles
```

If the dotfiles alias is not already configured in .zshrc, run

```bash
echo '\n# Dotfiles bare repo alias\nalias dotfiles="git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME"' >> ~/.zshrc
```

Activate the alias with

```bash
source .zshrc
```

Update your config files to the state tracked by the repository with

```bash
dotfiles restore
```

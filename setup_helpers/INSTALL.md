# Software to install manually before running setup.sh

## Shell
- [starship](https://starship.rs)

## Terminal
- [ghostty](https://ghostty.org)
- [tmux](https://github.com/tmux/tmux)

## Editor
- [neovim](https://neovim.io)
- [node](https://nodejs.org) — for LSP servers installed via mason
- [python 3](https://www.python.org) — venvs are set up by setup.sh
- [imagemagick](https://imagemagick.org) — for image rendering in neovim
- [fd](https://github.com/sharkdp/fd) — file finder used by telescope/snacks
- [ripgrep](https://github.com/BurntSushi/ripgrep) — grep used by telescope/snacks

## Java
- [openjdk](https://openjdk.org) — for lang.java, neotest-java, and MATLAB (macOS)

## Git
- [git](https://git-scm.com)
- [git-delta](https://github.com/dandavison/delta)
- [git-lfs](https://git-lfs.com)
- [lazygit](https://github.com/jesseduffield/lazygit)

## Dotfile management
- [stow](https://www.gnu.org/software/stow)

## PDF / Data science
- [miniconda](https://docs.conda.io/en/latest/miniconda.html)
- [sioyek](https://sioyek.info) — PDF viewer for vimtex
- [R](https://www.r-project.org) — also run `install.packages("languageserver")` in R
- jupytext — `pip install jupytext` or `conda install jupytext`

## macOS only
- [zsh-vi-mode](https://github.com/jeffreytse/zsh-vi-mode) — Linux is cloned by setup.sh
- [karabiner-elements](https://karabiner-elements.pqrs.org)
- [mactex](https://www.tug.org/mactex) — LaTeX distribution for vimtex

## Linux only
- [kanata](https://github.com/jtroo/kanata) — install via cargo
- [texlive](https://tug.org/texlive) — LaTeX distribution for vimtex

## Fonts
- [JetBrainsMono Nerd Font](https://www.nerdfonts.com)
